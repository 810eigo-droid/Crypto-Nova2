## 0. このプロジェクトは何か（30秒で）

Crypto Nova（暗号資産の市場データ分析ソフト／ライセンス385,000円）の集客を、制作・運用・改善でサポートする案件。
作るのは販売ページではなく、**「集客から販売までの導線（マシン）」**。

```
Meta広告 → リスト取りLP → 無料オファー登録（LINE/メアド）→ 公式LINEで教育・関係構築 → 販売
```

担当する成果物は、すべてこのマシンの「部品」。

- **成果保証型ではない**。評価は制作物・進行・改善提案を基準に。

### 成果報酬の交渉（結論：保留）

- 検討の結果、**実績非公開では割に合わないと判断し、成果報酬は一旦見送り**。当初の30万円の契約範囲で進めることに。

## 2. 戦略の核（絶対にぶらさない5軸）

1. **ポジショニング**：安い投資アプリではなく「知的な分析ライセンス」。プライベートバンク的な"静かな高級"。
2. **ターゲット**：準富裕層（年収800〜2,000万・投資余力あり・慎重派）。※今回のLP入口は「銀行とNISAに預けっぱなしの、堅実だが踏み出せていない層」に少し広げて設定。
3. **USP**：SNS＝主観／Crypto Nova＝客観データ。「データで"見る目"を持つ」。
4. **コンプラ**：投資助言・売買推奨・利益保証はしない。"過去データに基づく市場データ分析ソフト"という立ち位置を全媒体で守る。
5. **導線**：直接販売しない。広告→LP→登録→LINE教育→販売。

### 前回の失敗（重要な教訓）
- 前任者がMeta広告に40万円投下 → リスト80件（CPA5,000円）・成約1件のみ → クライアントが自主停止。
- 敗因＝**初心者・特典ハンターを集めてしまった（ターゲットと商品価格のミスマッチ）**。
- 対策＝今回は「クリックの量」でなく「払える人の質」を狙う。広告で初心者をふるい落とす。

## 3. トンマナ（デザインの正解・確定）

- **配色**：深緑（モスグリーン）×ゴールド。深緑＝信頼・成熟・本物。**グリーンは深く・くすませる**（鮮やかな緑は安っぽくなるので禁止）。
  - ベース `#0F2A22` / 暗部 `#0A1C16` / サブ `#2E4636` / ゴールド `#C8A24A` / 明ゴールド `#E3CE9A` / 文字 `#F4F1E8`。
  - ゴールドは面で塗らず、線・文字・縁取りで細く効かせる。
- **フォント**：見出し＝明朝（Shippori Mincho）、本文＝Zen Kaku Gothic New、英字アクセント＝Cormorant Garamond。
- **人物**：実写の上質モデル＋マスコット「ノヴァちゃん」併用。
- **禁止**：金貨・札束・高級車・ギラギラ＝射幸心演出（前回沈めた世界観）。**「静かな高級」を徹底**。
- ひとこと：**「煽らない。静かに、本物だけを見せる。」**

### マスコット「ノヴァちゃん」
- AI×サイボーグの知的な大人の女性。深緑のテック衣装＋ゴールドの差し色。こめかみに回路をうっすら。
- 試行錯誤の結論：幼すぎNG／色気・金ピカ過多NG →「知的で信頼できる大人のお姉さん」で確定。
- 現在3枚（正方形）：`ms-nova-front.webp`（正面・真顔）／`ms-nova-landscape.webp`（少し横）／`ms-nova-smaile.webp`（正面・笑顔）。
- 一貫性のコツ：マスター画像を参照添付し「同一人物・向きだけ変更」で生成。

## 4. メタ広告（確定・本番テスト用3本）

広告とLPは「最初の一行を揃える（メッセージマッチ）」が9割同じでOK。広告＝呼びかけ、LP＝その答え。

- **①本命**：本文「資産を、銀行とNISAに“預けっぱなし”になっていませんか。それは堅実。でも、それ“だけ”だと見えないものもあります。」／見出し「はじめての暗号資産ガイド（無料）」
- **②短縮**：本文「銀行とNISAに、預けっぱなし。——それ“だけ”で、本当に大丈夫でしょうか。」／見出し「何から始める？ 無料ガイド」
- **③一行**：本文「資産を、銀行とNISAに“預けっぱなし”のあなたへ。」／見出し「はじめての方の無料ガイド」

### バナー文言（ノヴァちゃん入り・深緑×ゴールド／文字は後乗せ）
- ①「銀行とNISAに、“預けっぱなし”ですか。」＋「はじめての暗号資産ガイド｜無料配布」
- ②「それ“だけ”で、大丈夫ですか。」＋「銀行・NISAの“次”を知る無料ガイド」
- ③「預けっぱなしのお金、ありませんか。」＋「はじめての方のための無料ガイド」

## 5. リスト取りLP（A案・確定）

- **入口（FV）**：「資産を、銀行とNISAに“預けっぱなし”のあなたへ。」← 変更しない。
- **流れ**：FV → 共感（踏み出せない自分の言語化）→ 視点（なんとなくで買わない人は何を見てるか）→ 無料オファー → 登録（LINE/メアド）→ Crypto Nova紹介 → クロージング。
- **無料オファー＝既存PDF（はじめての方向け 暗号資産スタートガイド）**。今あるPDFに正直に合わせて成立させた。
- B案（置き場所）・C案（情報の差）は**6/15納品後のA/Bテスト用にストック**。
- 実装：`index.html`（深緑×ゴールドで作成済み）。画像は`images/`の3枚。LINE URL・フォーム送信先は`[REPLACE]`で未設定。

### 無料オファーの将来計画（重要）
- 今は既存PDF（初心者向け・しょぼい）で成立させている。
- **6/15後の有料提案の目玉**として、「過去に大きく上昇したアルトに共通する“時系列パターン”の考え方」というデータ視点のプレゼントをWebページ化して差し替える。LPにその伏線（USPの種まき）は仕込み済み。
- ※これは契約30万を超える作業なので、有料提案として後出しする。今は無償で作り込まない。
